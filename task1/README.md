# Task 1: Social Network Friend Analysis System

## Project Overview
A command‑line social network simulator built with OOP principles. Users can register, add friends, and view friends. Future enhancements will include shortest‑path queries (BFS) and friend recommendations.

## Current Status (Pre‑submission)
- `User` class with basic attributes/methods.
- `Graph` class (undirected) with adjacency list and BFS.
- `SocialNetwork` class integrating users and the graph.
- Simple menu in `main.py` for registration and adding friends.

## Next Steps
- Implement shortest‑path using BFS.
- Implement friend recommendation based on common friends.
- Add input validation and data persistence.

## How to Run
Make sure you have Python 3 installed. Execute:
```bash
python main.py
```

## File Structure
- `user.py` – User class
- `graph.py` – Graph data structure
- `social_network.py` – Core logic
- `main.py` – Command‑line interface
